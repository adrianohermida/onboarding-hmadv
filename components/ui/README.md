# UI Kit Juridico HMADV

Biblioteca visual nativa do portal, mobile-first e compatível com cliente, advogado e administração.

Use `../styles/components.css` nas páginas para carregar o kit. Para comportamento interativo, o shell importa `components/ui/index.js`.

Componentes disponíveis:

- `ui-btn`
- `ui-input`
- `ui-select`
- `ui-textarea`
- `ui-modal`
- `ui-drawer`
- `ui-sidebar`
- `ui-card`
- `ui-table`
- `ui-tabs`
- `ui-timeline`
- `ui-steps`
- `ui-upload`
- `ui-badge`
- `ui-toast`
- `ui-calendar`
- `ui-dropdown`
- `ui-pagination`
- `ui-search`
- `ui-stat-card`
- `ui-kanban-card`
- `ui-form-section`

Padrões jurídicos:

- `ui-process-table`
- `ui-legal-timeline`
- `ui-document-upload`
- `ui-onboarding-steps`
- `ui-finance-card`
- `ui-client-panel`
- `ui-lawyer-panel`
