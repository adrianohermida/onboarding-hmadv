# Security Foundation (Sprint 0.4)

Enterprise security and LGPD foundation for Portal HMADV.

## Goals

- security-by-default for multi-tenant legal workflows
- auditable and traceable access and data flows
- consent and LGPD readiness
- secure document and biometric handling

## Absolute Rules

- no direct production mutation by AI
- no accidental public file access
- no cross-tenant access
- no document access without signed URL policy
- no sensitive-data exposure in logs or telemetry
