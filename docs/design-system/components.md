# Components

## Button
Use for primary and secondary actions.

## Input
Use for single-line textual input.

## Select
Use for constrained option sets.

## Checkbox
Use for independent boolean options.

## Radio
Use for mutually exclusive selections.

## Switch
Use for immediate toggle settings.

## Textarea
Use for longer text input.

## Card
Use for content grouping with elevation.

## Badge
Use for status and count highlights.

## Tooltip
Use for contextual helper hints.

## Dropdown
Use for compact action lists.

## Tabs
Use for local navigation between views.

## Accordion
Use for progressive disclosure.

## Progress
Use for task/journey progression.

## Skeleton
Use for loading placeholders.

## Alert
Use for inline informational feedback.

## Toast
Use for transient feedback messages.

## Avatar
Use for user identity display.

## EmptyState
Use when datasets or states are empty.
